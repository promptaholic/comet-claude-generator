## Frequently Asked Questions

### Q: Can I mix complexity levels?

**A:** Comet automatically chooses based on your prompt. For custom setups, start with the appropriate example and modify as needed.

### Q: How accurate are these examples?

**A:** These are real, unmodified outputs from Comet. Your results should match this quality.

### Q: Can I use these examples directly?

**A:** Yes! They're provided as starting templates. Customize for your specific use case.

### Q: What if my output doesn't match?

**A:** Check that:
1. You uploaded all 5 knowledge base files to Comet
2. Custom instructions are correctly pasted (1,442 chars)
3. Your prompt is specific enough
4. You're using the latest version

### Q: How do I generate my own examples?

**A:**
1. Set up Comet (see [QUICKSTART.md](../QUICKSTART.md))
2. Use a detailed, specific prompt
3. Copy all generated files
4. Test the output in a Claude Project
5. Verify it works first-try
